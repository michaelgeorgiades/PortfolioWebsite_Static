# PortfolioCircuit_Static

This is a static portfolio website showcasing professional experience, photography, and interactive games.

## Features

*   **Home Section:** Welcome message and introduction.
*   **Resume Section:** Details professional experience and technical skills.
*   **Photography Gallery:** Displays a collection of photographs with options to view and purchase.
*   **Games Section:** Features interactive games built with HTML5 Canvas (Snake, Tetris, Breakout, Pong).
*   **Contact Section:** A form to get in touch, integrated with reCAPTCHA.
*   **Live Chat:** Integrated with LiveChat for real-time communication.
*   **PayPal Integration:** For purchasing photographs.

## Project Structure

*   `.dockerignore`: Docker ignore file.
*   `.env.example`: Example environment variables.
*   `.gitignore`: Git ignore file.
*   `Dockerfile`: Docker configuration.
*   `favicon.ico`: Website favicon.
*   `gemini.md`: Gemini-specific markdown file (likely for internal documentation).
*   `github_action_key`, `github_action_key.pub`: GitHub Actions deployment keys.
*   `index.html`: Main HTML file.
*   `README.md`: Project README file (this file).
*   `styles.css`: Main CSS file.
*   `.github/workflows/deploy.yml`: GitHub Actions deployment workflow.
*   `attached_assets/`: Contains images and a PDF resume.
*   `js/`: Contains JavaScript files for various functionalities:
    *   `background.js`: Handles background animations.
    *   `contact.js`: Manages the contact form.
    *   `gallery.js`: Manages the photography gallery and modal.
    *   `games.js`: Contains game logic.
    *   `main.js`: Main JavaScript file for initialization and global functions.
    *   `navigation.js`: Handles navigation logic.
*   `shapeshift/`: A sub-project or distinct section with its own `index.html`, `script.js`, and `style.css`.
